## Build Executable File
- Change directory to the directory of `main.go` and `go.mod`.
- Build Executable Files
   - For server
     ```bash
     go build -i -v -o bin/server
     ```
   - For client
     ```bash
     go build -i -v -o bin/client
     ```
- Run Executable Files
   - For server
     ```bash
     ./bin/server
     ```
   - For client
     ```bash
     ./bin/client
     ```
